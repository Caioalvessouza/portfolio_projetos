XTREME GROOVY BIKES SALES Análise das principais métricas de Negócio

Identificação de preço venda e definição do preço ideal das motos da Empresa XTREME GROOVY BIKES SALES ajudando na expansão negócio, a partir de uma análise exploratória de dados em Python. Os dados fornecidos incluem informações cruciais sobre as motocicletas disponíveis, como fabricante, modelo, preço de venda, ano de fabricação, tipo de vendedor, histórico de proprietários, quilometragem percorrida, entre outros. O código disponibilizado utiliza a biblioteca Streamlit para criar uma interface interativa de análise de dados. Ele carrega o conjunto de dados e fornece respostas para questões importantes relacionadas às motocicletas disponíveis para revenda.

As ferramentas utilizadas foram:

  • Python, Pandas, Numpy
  • Github

  • Streamlit Cloud
